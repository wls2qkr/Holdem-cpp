# Holdem-cpp

게임 : 텍사스홀덤

핸드의 2장의 카드와 테이블의 5장의 카드를 조합해서 높은 족보를 가진 사람이 승리하는 게임

Game, User, Card 클래스 분할
Game : 게임 진행상황과 관련된 클래스
User : 참가자(이름, 가진 카드 등)
Card : 숫자와 모양으로 이루어진 카드클래스
